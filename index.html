Projects

AI Expert Chatbot

To experiment with advanced RAG techniques, i created a chatbot with a document lifecycle functionality.
For this project I used Langchain, Groq, and Trulens.
Whenever a document is added or deleted in the specified directory; the change is reflected in the AI's knowledge.
The application also has an admin panel to delete, view or add files.
It also provides evaluation metrics for every response.

Github Live Demo

Book Purchasing Assistant

This agentic application uses a Langraph (from the Langchain framework) and Groq as a provider to answer the user questions about books. The agent uses an LLM to determine what the next action should be (for example which tool to use, what does the user want to do, is the conversation over).

Github Live Demo

CineSense

CineSense is a sentiment analyzer for movies.
This project fine tuned an LLM over a movie reviews dataset to make it a sentiment analysis expert.
I got a chance to experiment with different types of LLMs and their capabilities, and also types of finetunings.
I also took this chance to experiment with hyperparameter tuning.

Github Live Demo

MumbaMate

MumbaMate was created for my Internship Capstone Project. It is a Slack app that makes HR support accessible and efficient for the company's employees using RAG (Retrieval Augmented Generation).
It also has a website that serves as an admin panel, allowing authorized individuals to manage files and view the statistical dashboard.

PhysioCare

PhysioCare was my Final Year Project developed as a team.
It is an AI-based physiotherapy assistant developed as a desktop-based application to guide you through your exercises.
PhysioCare leverages pose estimation and computer vision to ensure the correctness of exercises in real-time through a camera.
